# Interactive Periodic Table

An interactive and educational Periodic Table built using **HTML, CSS, and JavaScript**. This project is designed to make learning chemistry more engaging and intuitive.

## Features
- **Clickable elements**: View atomic number, symbol, and atomic mass for each element.
- **Color-coded groups**: Easily identify metals, non-metals, and metalloids.
- **Responsive design**: Works smoothly on both desktop and mobile devices.
- **Dynamic interactions**: Hover effects and animations make exploring fun and interactive.

## Technologies Used
- HTML
- CSS
- JavaScript

## Live Demo
Check out the interactive table here:

## Usage
1. Click on any element to view details.  
2. Hover over elements for a quick highlight effect.  
3. Explore different groups via color coding.
